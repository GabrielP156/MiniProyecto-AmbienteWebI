# GESTATAREA - Mini Proyecto Web

## Descripción
GESTATAREA es una aplicación web desarrollada con HTML, CSS y JavaScript que permite gestionar tareas académicas de forma sencilla. Permite crear, editar, eliminar y organizar tareas con fechas límite, prioridades y estados.

El proyecto tiene como objetivo practicar el manejo del DOM, eventos en JavaScript y almacenamiento de datos en el frontend.

## Funcionalidades

- Crear tareas
- Editar tareas
- Eliminar tareas
- Listar tareas por estado (pendiente, en progreso, completada)
- Filtrar tareas por fecha o prioridad
- Gestión de fechas límite
- Cálculo de porcentaje o puntaje por tarea
- Almacenamiento en LocalStorage

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
